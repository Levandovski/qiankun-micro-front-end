## Getting Started with qiankun micro front end framework

### Project adapted for version 18 of react

### In this project I configure qiankun integration with all types below

#### main-react-app =>>> Main-App (JS)

https://create-react-app.dev/

#### micro-react-sub-app1 =>>> Sub-App-Micro (JS) (CRA)

https://create-react-app.dev/

#### micro-react-sub-app2 =>>> > Sub-App-Micro (JS) (CRA)

https://create-react-app.dev/

#### micro-react-sub-app3-ts-vite =>>> > Sub-App-Micro (Typescript) (Vite)

https://vitejs.dev/

#### micro-react-sub-app4-ts-cra =>>> > Sub-App-Micro (Typescript) (CRA)

https://create-react-app.dev/docs/adding-typescript/

### Installing all dependencies

#### npm install

### Start all projects

#### npm start

### Attention this configuration below are for CRA application (Create React App)

#### To overwrite webpack you need lib rewrite =>>> > react-app-rewired

https://www.npmjs.com/package/react-app-rewired

### As vite does not use webpack it is necessary to use a plugin. The plugin used for this is vite-plugin-qiankun

https://github.com/tengmaoqing/vite-plugin-qiankun

##### For more details visit the official qiankun documentation link

https://qiankun.umijs.org/
